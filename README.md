# **ОБЛАЧНОЕ ХРАНИЛИЩЕ MyCloud**
## Пользовательский интерфейс (Frontend)<br>(разработан с использованием React)

### 1. *Структура проекта*

`d_fr/` - корневая папка фронтенда<br>
&nbsp;&nbsp;`src/` - папка с исходным кодом приложения<br> 
&nbsp;&nbsp;&nbsp;&nbsp;`assets/` – папка с иконками для интерфейса<br> 
&nbsp;&nbsp;&nbsp;&nbsp;`components/` - папка с UI-компонентами<br>
&nbsp;&nbsp;&nbsp;&nbsp;`pages/` – папка со страницами приложения<br> 

`package.json` - файл с зависимостями и скриптами для сборки<br>
`README.md` - файл с инструкцией по установке и развертыванию 

### 2. *Развертывание проекта на локальном компьютере*

1. Запускаем терминал в директории, где планируется разместить папку с файлами проекта
   
2. Клонировать репозиторий:
   `git clone https://github.com/ART20230129/d_fr`

3. Перейти в папку `d_fr/`
   
   ```bash
   cd d_fr
   ```

4. Установить необходимые зависимости
   
   ```bash
   npm i
   ```

5. В файле config.js  в папке `d_fr\scr\` прописать
   `API_BASE_URL = 'ваш_url'` (например, 'http://localhost:8000' или адрес ip адрес сервера)

6. Запустить одним из способов:

- в режиме разработки:

   ```bash
   npm run dev
   ```

- в режиме предварительного просмотра собранного приложения:

   ```bash
   npm run build
   ```
